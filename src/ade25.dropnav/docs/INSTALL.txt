ade25.dropnav Installation
--------------------------

To install ade25.dropnav using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ade25.dropnav`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ade25.dropnav
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
